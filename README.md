# Mineplace3D [Beta]

Mineplace3D is a free and open-source voxel-based sandbox game inspired by Minecraft. It is developed by a team of passionate friends who aim to create a fun and engaging experience without the influence of corporate greed.

## Contributing

We welcome contributions from the community! If you're interested in helping out with development, bug fixes, or new features, please join our [Discord server](https://discord.gg/FZ4H6tmfRs) to get involved.

## Download

You can download the latest version of the Mineplace3D Launcher [here](https://github.com/Muhtasim-Rasheed/mineplace3d-launcher/releases). The launcher will help you install and manage the game easily. The latest stable version of Mineplace3D is `0.3.1-alpha` (Sorry, we haven't finished beta yet!).
